Documentation for Café Gupshup's Bot

Introduction

This page documents features of ##gupshup's bot, ChaiWala ("tea-server"). ChaiWala runs a modified version of the IRC bot CloudBot. Amongst the modifications made to ChaiWala include a virtual currency system, described in more detail below.

Note that the full list of commands for ChaiWala is not given below; for that, you should consult the Snoonet page on gonzobot, which is a version of CloudBot customised for Snoonet. Some commands listed on that page may not be present in ChaiWala because ChaiWala runs a slightly older version of CloudBot. If you would like to see ChaiWala be updated with a particular command, contact hasc on ##gupshup.

In this documentation, command parameters are surrounded by angle brackets ("<" and ">") to indicate that they are compulsory, and square brackets if optional.

The Virtual Currency System

A rudimentary virtual currency system has been implemented into ChaiWala. It allows users to exchange a currency known as coins. At the moment, the only thing you may do with coins is purchase the removal of your own grabs (for a cost of 10 coins per grab, as documented below), but soon you will be able to trade ducks and other items for coins. Coins can be obtained by participating in the channel. Each time you post a message to the channel, there is a 1 in 1000 chance that the bot dishes out a coin to you.

List of Commands

Syntax Description
.openaccount Opens an account in your name, after which you will be able to hold coins and exchange them with other users.
.balance [nick] Shows [nick]'s current balance, in coins. If [nick] is left blank, this will simply display your own balance.
.coinstotal, .totalcoins Shows the total number of coins currently in circulation. This is the sum of all coins possessed by users.
.give <nick> <quantity> Transfers <quantity> coins from your balance into <nick>'s balance. This will not work if <nick> does not have an account (the command will display a message indicating this). Channel operators may specify a negative value for <quantity> to take coins from a particular nick.
.richest Shows the richest nicks in the channel: a list of all nicks with accounts and their corresponding balances sorted in descending order.

Markov Chain Generator

You can have the bot feed all previous messages in the channel from a particular nick into a Markov chain generator and then output a random message from it. To make use of this feature, simply type <nick>_bot (e.g. hasc_bot) and the bot will respond with a randomly generated message. You may also type ?_bot to pick a random nick, or grabs_bot to feed all previous grabs into the Markov chain generator instead.

Miscellaneous Commands

Use .platts <word> to consult Platts' dictionary for a particular Urdu word. You can type the word in Roman Urdu but it tends to work best if you write it in the Urdu script. By default, the first definition will be returned but you can view subsequent definitions by adding a space followed by a number, as so: .platts <word> 2.

Use .cricket to see the latest cricket scores. The code for this command was written a long time and might take a long time to return its output. Sometimes it might not work altogether, so tread carefully.

Some other fun1 commands include, but are not limited to, .kek, .nep, and .konishiki.
1 Definitions of "fun" may vary.
